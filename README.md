# dotfile

## 目录结构

```bash
- home  # 模拟 Linux 的 home 目录
    - .config   # XDG 风格的配置目录
        - tmux  # tmux 的配置文件
        - nvim  # nvim 配置
    - .zshrc    # zsh  的配置文件
```

> 终端优化的很大一部分工作都是字体和符号的优化
